# Hybrid Neural Network with Statistical Forecasting
### `Make sure you are using Python 3.7.3 as Tensorflow does not support support Python 3.8`

- Install needed libraries
- In Terminal, you should be inside /model directory and run thr script with the dataset you want

`(directory)\model> python <dataset filename>`

Example:
```sh
$ python hwlstm.py 6_134_1-1-2013_12-31-2018.json
```


*Note: dataset was scraped from https://edge.pse.com.ph/. you can use your own univariate data following the same json structure*
